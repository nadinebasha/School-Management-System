# School Management System

A Python-based desktop application for managing academic records, schedules, grading, and school administration tasks.

---

## Features
- Student record management
- Grading system
- Scheduling workflows
- Feedback management
- Object-oriented application design

---

## Technologies Used
- Python
- JSON
- OOP

---
## Functionalities
- Add and manage students
- Store academic records
- Generate grading workflows
- Administrative task handling
